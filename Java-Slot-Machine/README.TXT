------------------------------------------------------------------------
Simple 1 line slot machine written in Java. Start with a bankroll of 100
and allow user bets in increments of: 1, 2, 5, 10, 25, 50. Green button 
begins slot machine animation. Arrows increase/decrease bet size. 
Any 2 matching fruit wins bet amount.
All 3 matching fruit wins 3x bet amount.
------------------------------------------------------------------------
